# AutoSOS Frontend

Frontend aplikacji AutoSOS - React PWA dla użytkowników potrzebujących pomocy drogowej.

## Instalacja

```bash
npm install
```

## Uruchomienie w trybie deweloperskim

```bash
npm run dev
```

Aplikacja będzie dostępna pod adresem `http://localhost:5173`

## Build produkcyjny

```bash
npm run build
```

Pliki gotowe do wdrożenia znajdą się w folderze `dist/`.

## Funkcjonalności

- ✅ Formularz wezwania pomocy z wyborem lokalizacji A i B
- ✅ Integracja z mapą Leaflet (OpenStreetMap)
- ✅ Lista dostępnych operatorów z cenami
- ✅ PWA support (service worker, manifest)
- ✅ Optymalizacja dla słabych sieci (caching)

## Technologie

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Leaflet / OpenStreetMap
- vite-plugin-pwa
